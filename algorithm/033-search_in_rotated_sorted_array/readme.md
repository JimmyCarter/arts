# 在旋转排序数组中查找

设：一个通过升序排序的数组被旋转了，转轴是你事先不知道的。

(比如：[0,1,2,4,5,6,7]可能变成了[4,5,6,7,0,1,2])。

会给你一个目标值用于搜索，如果能够在数组中找到，就返回它的索引，否则就返回-1。

你可以假设数组中没有重复元素。

你的算法时间复杂度必须在O(log n)以内。

```
Example 1:
Input: nums = [4,5,6,7,0,1,2], target = 0
Output: 4

Example 2:
Input: nums = [4,5,6,7,0,1,2], target = 3
Output: -1
```
